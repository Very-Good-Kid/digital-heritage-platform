# 数字遗产继承平台 - 网站使用说明

## 🌐 这是一个网站，不是APP

本产品是一个基于Web技术的**网站**，通过浏览器访问，无需下载安装任何APP。

---

## 🚀 如何访问网站

### 方法一：本地访问（推荐用于测试）

1. **启动网站服务器**
   ```bash
   python app.py
   ```
   或直接双击 `start.bat` 文件

2. **在浏览器中打开**
   - 打开Chrome、Firefox、Edge等浏览器
   - 在地址栏输入：`http://localhost:5000`
   - 按回车键访问

### 方法二：局域网访问（手机、平板访问）

1. **启动网站服务器**（同上）

2. **查看电脑IP地址**
   - Windows：打开命令提示符，输入 `ipconfig`
   - 找到"IPv4 地址"，例如：`192.168.1.7`

3. **在手机浏览器中访问**
   - 确保手机和电脑在同一WiFi网络
   - 在手机浏览器输入：`http://192.168.1.7:5000`
   - 按回车访问

### 方法三：公网访问（部署后）

部署到云服务器后，任何人都可以通过域名访问：
- 例如：`www.yourdomain.com`

## 🛠️ 常见问题

### Q1: 网站打不开怎么办？
**A:**
1. 检查网站服务器是否启动（运行 `python app.py`）
2. 检查浏览器地址是否正确：`http://localhost:5000`
3. 尝试使用其他浏览器
4. 检查防火墙设置

### Q2: 手机如何访问？
**A:**
1. 确保手机和电脑在同一WiFi网络
2. 查看电脑IP地址（`ipconfig`）
3. 在手机浏览器输入：`http://电脑IP:5000`

### Q3: 可以让别人访问我的网站吗？
**A:**
- 局域网内：可以，通过IP地址访问
- 公网：需要部署到云服务器，配置域名

### Q4: 网站数据保存在哪里？
**A:**
- 本地开发：保存在 `digital_heritage.db` 文件中
- 生产环境：保存在数据库服务器中

### Q5: 如何停止网站？
**A:**
- 在运行 `python app.py` 的命令行窗口按 `Ctrl+C`
