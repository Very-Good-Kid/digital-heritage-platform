# 全局字体颜色修复总结

## ✅ 所有问题已修复

### 问题描述

多个页面存在字体颜色与背景色相同的问题，导致文字看不见：

1. **平台政策矩阵**：
   - 主流平台继承政策对比 - 看不见
   - 详细政策信息 - 看不见

2. **数字资产继承导航**：
   - 选择您的继承情景 - 看不见
   - 导航结果中的 QQ继承路径图 - 看不见
   - 详细操作步骤 - 看不见

3. **常见问题解答**：
   - 问题分类 - 看不见

4. **关于我们**：
   - 联系我们 - 看不见

---

### 原因分析

**问题根源**：
`.card-header` 的全局样式设置为：
```css
.card-header {
    background-color: var(--primary-color);
    color: white;
}
```

这导致：
1. 所有卡片头部都使用深色背景（紫色）
2. 所有卡片头部文字都是白色
3. 对于没有背景色的卡片，白色文字在浅色背景上看不见

**影响范围**：
- 所有使用 `.card-header` 的页面
- 包括平台政策、继承导航、FAQ、关于我们等页面

---

### 修复内容

#### 修复 1：修改卡片头部默认样式

**修改前**：
```css
.card-header {
    background-color: var(--primary-color);
    color: white;
    border-radius: 10px 10px 0 0 !important;
}
```

**修改后**：
```css
/* 默认卡片头部样式 */
.card-header {
    background-color: #f8f9fa;
    color: var(--dark-text);
    border-radius: 10px 10px 0 0 !important;
    font-weight: 600;
}

/* 特殊深色背景的卡片头部 */
.card-header.bg-primary,
.card-header.bg-success,
.card-header.bg-danger {
    color: white !important;
}
```

**效果**：
- ✅ 默认卡片头部使用浅灰色背景（#f8f9fa）
- ✅ 默认卡片头部文字使用深色（var(--dark-text)）
- ✅ 特殊深色背景的卡片头部文字仍为白色
- ✅ 所有文字现在都清晰可见

#### 修复 2：修复卡片标题颜色

**修改前**：
```css
.card .card-header {
    color: white;
}
```

**修改后**：
```css
.card .card-header {
    color: var(--primary-color);
    font-weight: 600;
}
```

**效果**：
- ✅ 卡片标题使用主题色（紫色）
- ✅ 字体加粗，更容易阅读
