# 数字遗产继承平台 - 项目总结

## 项目概述

数字遗产继承平台是一个完整的Web应用程序，旨在为用户提供数字遗产规划和继承服务。项目按照V1.0需求文档完成开发，实现了核心功能闭环。

## 技术实现

### 技术栈
- **后端框架**：Flask 3.1.2
- **数据库**：SQLite（支持扩展到MySQL/PostgreSQL）
- **ORM**：Flask-SQLAlchemy 3.1.1
- **用户认证**：Flask-Login 0.6.3
- **表单处理**：Flask-WTF 1.2.2
- **加密技术**：cryptography 46.0.3
- **PDF生成**：reportlab 4.4.9
- **图像处理**：Pillow 11.2.1
- **前端框架**：Bootstrap 5.3
- **图标库**：Bootstrap Icons 1.11.0

### 数据库设计

**7个数据表：**
1. users - 用户表
2. digital_assets - 数字资产表
3. digital_wills - 数字遗嘱表
4. platform_policies - 平台政策表
5. stories - 故事表
6. faqs - FAQ表
7. 继承关系已通过外键建立

**数据关系：**
- User → DigitalAssets (一对多)
- User → DigitalWills (一对多)
- DigitalWill → DigitalAssets (多对多，通过JSON字段)
